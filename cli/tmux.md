# tmux notes

Prefix remapped to Ctrl-a

### Create a new session

tmux new -s newsession

### Detach session

prefix + d

### Attach session

tmux attach -t session 

### Split horizontally

prefix + %

### Split vertically

prefix + "

### Windows

prefix + c

### Pane Navigation

up    : Ctrl-b uparrow
down  : Ctrl-b downarrow
left  : Ctrl-b leftarrow
right : Ctrl-b rightarrow
